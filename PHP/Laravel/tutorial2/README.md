## Docker環境
出典：[ucan-lab/docker-laravel](https://github.com/ucan-lab/docker-laravel)
### バージョン
- PHP：7.4.14
- Laravel：5.8.19
- MySQL：8.0.23

### 初回環境構築
.env ファイル作成
```
$ cp .env.example .env
```

初回セットアップ
```
$ make init
```

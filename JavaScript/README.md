## JavaScript
動的型付けのスクリプト言語。

フロントのHTMLページにクライアント側のふるまいを持たせることができる。Webページを構成するオブジェクトを操作可能。

また、サーバーサイドとしてのNode.jsがある。

[公式リファレンス](https://developer.mozilla.org/ja/docs/Web/JavaScript/Reference)

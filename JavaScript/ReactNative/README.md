## よく使うコマンドメモ
- Expoサーバ起動  
  `$ expo start`  
  オフライン時は`$ expo start --offline`

- エミュレータデバイスの確認  
  `$ emulator -list-avds`

- 指定デバイスのエミュレータ起動  
  `$ emulator -avd (デバイス名)`

- エミュレータのDeveloper Tool起動  
  `ctrl + D`のメニューから`Toggle Inspector`

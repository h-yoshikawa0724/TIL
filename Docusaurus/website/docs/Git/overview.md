## 概要
ファイルのバージョン管理システム。
- 最新のバージョンが分かる
- いつ・だれが・何を 編集したのかわかる
- 前のバージョンにも戻せる

ホスティングサービスにはいかのようなものがある。
- GitHub
- GitLab
- BitBucket
